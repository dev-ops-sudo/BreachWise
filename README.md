# BreachWise

Interactive cybersecurity incident response simulator powered by real-time injects and AI mentor guidance.

## Key Features

- **Interactive Incident Scenarios**: Engage in simulated cybersecurity scenarios (Ransomware, Phishing, DDoS, Insider Threat, and more).
- **Real-Time AI Guidance**: Get real-time advice and solution breakdowns using AI Mentor mode.
- **Performance Evaluation**: Receive comprehensive rankings, readiness levels, strengths/weaknesses mapping, and custom recommendations.
- **NIST Alignment**: Standardized mapping to incident response steps and guidelines.

## Tech Stack

- **Framework**: Next.js & React (TypeScript)
- **Database / Auth**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS
- **AI Engine**: Google Gemini / Anthropic SDK
